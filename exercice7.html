Exercice 7 – Transitions et animations

7.1 – Bouton animé au survol

↓ Survolez ou cliquez sur le bouton ↓

Code CSS pour le bouton animé :

/* L'état normal du bouton */
.bouton {
    background-color: #4a90d9;
    color: white;
    border: none;
    padding: 14px 30px;
    border-radius: 6px;
    font-size: 16px;
    cursor: pointer;
    transition: background-color 0.3s ease,
                transform 0.2s ease,
                box-shadow 0.2s ease;

    box-shadow: 0 3px 10px rgba(74, 144, 217, 0.4);
}

/* Au survol (hover) */
.bouton:hover {
    background-color: #357abd;   
    transform: translateY(-4px); 
    box-shadow: 0 8px 20px rgba(74, 144, 217, 0.6);
}

/* Au moment du clic */
.bouton:active {
    transform: translateY(0);  
}
transition crée l'effet de glissement entre l'état normal et l'état :hover. Sans transition, le changement serait instantané et grossier.

7.2 – Spinner de chargement avec @keyframes

↑ Survolez cette carte aussi !

Code CSS pour le spinner :

/* ÉTAPE 1 : Définir l'animation avec @keyframes */
@keyframes tourner {
    from { transform: rotate(0deg); }   
    to   { transform: rotate(360deg); } 
}

/* ÉTAPE 2 : Créer le cercle */
.spinner {
    width: 50px;
    height: 50px;
    border: 5px solid #e0e0e0;    
    border-top-color: #4a90d9;     
    border-radius: 50%;            
    animation: tourner 0.8s linear infinite;
}
/* Bonus : animation de texte avec opacité */
@keyframes clignoter {
    0%   { opacity: 1; }
    50%  { opacity: 0.3; }
    100% { opacity: 1; }
}

.texte-chargement {
    animation: clignoter 1.5s ease-in-out infinite;
}
@keyframes définit les étapes de l'animation. animation: applique cette animation à un élément. Les 4 valeurs sont : nom, durée, courbe (linear/ease), répétition (infinite = infini).
