Exercice 2 – Box Model

2.1 – Comprendre le Box Model

Le Box Model CSS est composé de 4 couches :

Margin (espace extérieur)
Border (bordure)
Padding (espace intérieur)
Contenu (width / height)

1. Avec box-sizing: content-box (défaut) — quelle est la largeur totale visible ?

/* La boite de l'exercice : */
.box {
    width: 300px;
    padding: 20px;
    border: 5px solid #333;
    margin: 15px;
}


/* Calcul de la largeur visible :
   width   = 300px (contenu seulement)
   padding = 20px gauche + 20px droite = +40px
   border  = 5px gauche + 5px droite  = +10px
   TOTAL   = 300 + 40 + 10 = 350px visible
   (le margin n'est pas "visible", il crée de l'espace autour) */

Démo avec content-box (largeur visible = 350px) :

Je fais 300px de contenu, mais 350px visible avec padding+border
Réponse : 350px (300 de contenu + 40 de padding + 10 de bordure)

2. Avec box-sizing: border-box — quelle est la largeur totale visible ?

/* Avec border-box, le padding et border sont inclus dans width */
.box {
    box-sizing: border-box;
    width: 300px;
    padding: 20px;
    border: 5px solid #333;
    margin: 15px;
}

/* Calcul :
   Largeur visible = 300px (tout compris !)
   Largeur du contenu = 300 - 40 (padding) - 10 (border) = 250px
*/

Démo avec border-box (largeur visible = 300px) :

Je fais exactement 300px visible, padding+border inclus
Réponse : 300px. C'est beaucoup plus simple à calculer !

3. Le reset CSS universel pour box-sizing

/* On met ce code AU DEBUT de tout fichier CSS */
*, *::before, *::after {
    box-sizing: border-box;
}
L'astérisque (*) signifie "tous les éléments". Ainsi, tous les éléments de la page auront box-sizing: border-box. Les ::before et ::after sont des pseudo-éléments qu'on inclut aussi.

2.2 – Centrer un élément

Centrer horizontalement avec margin: auto

/* Pour centrer horizontalement, l'élément doit avoir une largeur définie */
.ma-div {
    width: 600px;
    margin: 0 auto;  /* 0 en haut/bas, auto à gauche ET à droite */
}
Je suis centré horizontalement !
margin: 0 auto : auto divise l'espace restant en deux parts égales à gauche et à droite. Résultat = centré !

Centrer horizontalement ET verticalement avec Flexbox

/* Sur l'élément PARENT */
.conteneur {
    display: flex;
    justify-content: center;  /* centrer sur l'axe horizontal */
    align-items: center;      /* centrer sur l'axe vertical */
    min-height: 100vh;        /* prend toute la hauteur de la page */
}

/* L'élément enfant n'a besoin que d'une largeur */
.ma-div {
    width: 600px;
}
Je suis centré dans les deux sens !
Flexbox est la méthode moderne recommandée pour centrer. Le parent est le "conteneur flex", l'enfant se centre automatiquement.
