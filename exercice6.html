Exercice 6 – Variables CSS et thèmes

Les variables CSS permettent de définir des valeurs réutilisables et de créer des thèmes facilement.

☀️ Thème clair actif
Cette carte change de couleur avec le thème !

Toutes les couleurs viennent des variables CSS. Quand on change le thème, toute la page s'adapte automatiquement.

Le code CSS complet :

/* ====== VARIABLES dans :root = disponibles partout ====== */
:root {
    --couleur-fond:    #f5f5f5;
    --couleur-texte:   #333333;
    --couleur-accent:  #4a90d9;
    --couleur-carte:   #ffffff;
}

/* ====== THÈME SOMBRE — on redéfinit les variables ====== */
[data-theme="sombre"] {
    --couleur-fond:    #1a1a2e;
    --couleur-texte:   #e0e0e0;
    --couleur-accent:  #0f3460;
    --couleur-carte:   #16213e;
}

body {
    background-color: var(--couleur-fond);
    color: var(--couleur-texte);
    transition: background-color 0.3s; 
}

.carte {
    background-color: var(--couleur-carte);
}

h1, h2 {
    color: var(--couleur-accent);
}
/* ====== JavaScript pour changer le thème ====== */
function changerTheme() {
    const body = document.body;
    const themeActuel = body.getAttribute('data-theme');

    if (themeActuel === 'sombre') {
        body.removeAttribute('data-theme'); 
    } else {
        body.setAttribute('data-theme', 'sombre'); 
    }
}
Les variables CSS commencent toujours par deux tirets (--). On les déclare dans :root (= élément racine = <html>) pour qu'elles soient accessibles partout.
