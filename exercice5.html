Exercice 5 – Responsive Design (Mobile First)

Mobile First = on écrit d'abord le CSS pour mobile, puis on ajoute des media queries pour les écrans plus grands.

Démo — Réduisez la fenêtre pour voir le responsive en action !

Navigation :

  • Accueil
  • Services
  • Contact
Sidebar
Visible sur tablette et desktop
Contenu principal
Sur mobile, la sidebar est cachée et le contenu prend toute la largeur.

Code CSS complet (Mobile First) :

 
.container {
    width: 100%;
    padding: 15px;
}

.nav-menu {
    display: flex;
    flex-direction: column; 
    gap: 5px;
}

.content  { display: block; } 
.sidebar  { display: none; }   


@media (min-width: 768px) {
    .container   { max-width: 90%; margin: 0 auto; }
    .nav-menu    { flex-direction: row; gap: 15px; } 
    .sidebar     { display: block; width: 200px; }  
    .content     { display: flex; gap: 1rem; }       
}


@media (min-width: 1024px) {
    .container   { max-width: 1200px; }
    .sidebar     { width: 280px; }
}
Règle Mobile First : on utilise min-width dans les media queries (pas max-width). On écrit le CSS mobile sans media query, puis on "améliore" pour les grands écrans.
