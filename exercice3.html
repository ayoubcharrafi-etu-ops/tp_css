Exercice 3 – Flexbox

Flexbox permet d'aligner des éléments sur une ligne (ou une colonne) très facilement.

3.1 – Barre de navigation

MonSite
  • Accueil
  • Services
  • Contact

Code CSS pour la barre de navigation :

/* Le conteneur principal de la navbar */
.navbar {
    display: flex;                   
    align-items: center;             
    justify-content: space-between;  
    padding: 12px 20px;
    background-color: #333;
}

.nav-links {
    list-style: none;
    display: flex;     
    gap: 20px;         
    margin: 0;
    padding: 0;
}

.nav-links a {
    color: #ccc;
    text-decoration: none;
}
justify-content: space-between pousse le premier enfant (logo) complètement à gauche et le dernier (nav-links) à droite. C'est la propriété clé !

↑ Remarquez que les 3 boutons sont tous alignés en bas, même avec des contenus de longueurs différentes !

Code CSS pour les cartes :

/* Conteneur des cartes */
.cards {
    display: flex;   
    gap: 15px;       
}

.card {
    flex: 1;                   
    display: flex;             
    flex-direction: column;    
    padding: 15px;
    border: 1px solid #ddd;
}

.card p {
    flex: 1;  
}

.card-link {
    margin-top: auto;  
}
Astuce clé : On met display: flex; flex-direction: column sur la carte, puis flex: 1 sur le paragraphe. Le paragraphe "grandit" pour remplir l'espace, ce qui pousse le bouton vers le bas dans toutes les cartes.
