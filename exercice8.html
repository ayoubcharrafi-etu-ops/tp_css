Exercice 8 – Quiz de révision

Les bonnes réponses sont en vert ✅. Les mauvaises sont barrées.

1 Quel sélecteur a la spécificité la plus haute ?

a) .header .nav a   → score : 0,2,1
✅ b) #header a   → score : 1,0,1 (le plus fort !)
c) header nav a   → score : 0,0,3
d) a.active   → score : 0,1,1
Pourquoi b ? Le # (ID) vaut 1,0,0. Un ID bat toujours une classe ou une balise. Donc #header a (1,0,1) est plus fort que .header .nav a (0,2,1).

2 Avec box-sizing: border-box, un élément avec width: 200px et padding: 20px a une largeur totale de :

a) 240px   → c'est le résultat sans border-box
✅ b) 200px   → border-box inclut le padding dans width
c) 220px
d) 160px
Pourquoi b ? Avec border-box, la largeur totale reste toujours égale à ce qu'on a écrit : 200px. Le padding (20px x 2 = 40px) est déduit de l'intérieur, donnant 160px de contenu, mais la largeur visible reste 200px.

3 Pour centrer un élément verticalement et horizontalement avec Flexbox :

a) justify-content: center seulement → centre seulement sur l'axe horizontal
b) align-items: center seulement → centre seulement sur l'axe vertical
✅ c) justify-content: center ET align-items: center
d) text-align: center → ne centre pas un bloc, seulement le texte
Pourquoi c ? Il faut les deux propriétés car Flexbox a deux axes : horizontal (justify-content) et vertical (align-items). Pour être centré dans les deux sens, il faut agir sur les deux axes.

4 La valeur 1fr en CSS Grid signifie :

a) 1 pixel
✅ b) 1 fraction de l'espace disponible
c) 1 pourcentage
d) 1 rem
Pourquoi b ? fr vient de "fraction". Par exemple, grid-template-columns: 1fr 2fr crée deux colonnes : la première prend 1/3 de l'espace, la deuxième prend 2/3. C'est proportionnel à l'espace disponible.

5 L'approche "Mobile First" utilise :

a) max-width → c'est l'approche "Desktop First" (l'inverse)
✅ b) min-width dans les media queries
c) device-width
d) orientation
Pourquoi b ? Mobile First = on commence par coder pour mobile (sans media query), puis on ajoute @media (min-width: 768px) pour adapter aux tablettes et @media (min-width: 1024px) pour les desktops. On va du petit vers le grand.
